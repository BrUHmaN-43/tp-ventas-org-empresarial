## Escenario Elegido
**Escenario B – Análisis de Ventas de una Pequeña Empresa**

## Dataset Utilizado
Dataset simulado de ventas comerciales con registros diarios.  
Fuente: https://gist.github.com/khanusama20/ee33c2869dd5cf3cebdf020be1ca43f6

## Descripción del Proyecto
Script Python que procesa datos de ventas para calcular:
- Ventas totales
- Producto más vendido
- Ventas por mes
- Gráfico de evolución de ventas

## Estructura del Repositorio

tp-ventas-org-empresarial/
├── datos/          → Dataset CSV de ventas
├── scripts/        → Script Python de análisis
├── resultados/     → Gráficos y resultados exportados
├── README.md
└── .gitignore

## Instrucciones para Ejecutar
1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar `scripts/analisis_ventas.py`
4. Los resultados se guardan automáticamente en `/resultados`
